# Autonomous Sales Intelligence Agent

An AI-powered sales intelligence platform built using Python, Streamlit, Prophet, and Google Gemini.

## Features

- Dataset Profiling
- Sales Analytics
- KPI Dashboard
- Sales Forecasting
- AI Insights
- Business Recommendations
- Automated Report Generation

## Technologies

- Python
- Pandas
- Plotly
- Streamlit
- Prophet
- Google Gemini
- Agentic AI

## Project Architecture

Dataset Upload
→ Profiling Agent
→ Sales Analytics Agent
→ Forecast Agent
→ AI Insight Agent
→ Recommendation Agent
→ Report Agent

```bash
streamlit run app.py
